Homepage Route

Route: /

This page represents the root route of the static learning project. Later, an AWS Application Load Balancer will send requests for / to a dedicated target group and EC2 instance.

What this lab demonstrates

Each section of the site is intentionally split into its own route so you can practice path-based routing concepts without backend code.

  • / is the landing page.
  • /trending will model a trending content service.
  • /series will model a series catalog service.
  • /movies will model a movie catalog service.
  • /new will model a new releases service.

Infrastructure Note

This page will eventually be served by its own EC2 instance behind an AWS Application Load Balancer target group.

Static HTML and CSS only. AWS deployment will be added later.
